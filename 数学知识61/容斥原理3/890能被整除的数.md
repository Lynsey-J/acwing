### 题目描述

给定一个整数  n  和  m  个不同的质数  p1,p2,…,pm 。

请你求出  1∼n  中能被  p1,p2,…,pm  中的至少一个数整除的整数有多少个。

输入格式
第一行包含整数  n  和  m 。

第二行包含  m  个质数。

输出格式
输出一个整数，表示满足条件的整数的个数。

数据范围
1≤m≤16 ,
1≤n,pi≤10^9

### 样例

Input

```
10 2
2 3
```

Output

```
7
```
