### 题目描述

给定一个  n×m  的网格，请计算三点都在格点上的三角形共有多少个。

下图为  4×4  的网格上的一个三角形。

a.png

注意：三角形的三点不能共线。

输入格式
输入一行，包含两个空格分隔的正整数  m  和  n 。

输出格式
输出一个正整数，为所求三角形数量。

数据范围
1≤m,n≤1000

### 样例

Input

```
2 2
```

Output

```
76
```
