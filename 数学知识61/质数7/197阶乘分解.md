### 题目描述

给定整数  N ，试把阶乘  N!  分解质因数，按照算术基本定理的形式输出分解结果中的  pi  和  ci  即可。

输入格式
一个整数  N 。

输出格式
N!  分解质因数后的结果，共若干行，每行一对  pi,ci ，表示含有  pcii  项。按照  pi  从小到大的顺序输出。

数据范围
1≤N≤10^6

### 样例

Input

```
5
```

Output

```
2 3
3 1
5 1
```

#### 样例解释
5!=120=23∗3∗5
